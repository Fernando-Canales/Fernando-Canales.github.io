Sobre mí

Soy un Analista de Datos y Desarrollador Python con un Doctorado en Astrofísica. Me especializo en extraer conclusiones de negocio a partir de conjuntos de datos masivos y complejos, utilizando Python, pruebas estadísticas avanzadas y herramientas modernas de visualización de datos.

Durante mi doctorado, realizado entre el Observatorio de París y el Instituto Max Planck, desarrollé algoritmos de detección de anomalías de misión crítica para la misión espacial PLATO de la Agencia Espacial Europea. Este riguroso entorno académico me inculcó una profunda capacidad para resolver problemas altamente ambiguos, diseñar pipelines de datos robustos y escribir código listo para producción.

Ahora estoy llevando esta potencia analítica al sector tecnológico corporativo. Ya sea construyendo dashboards interactivos en Tableau, desarrollando paquetes de Python de código abierto o utilizando pruebas A/B para optimizar operaciones de negocio, mi enfoque está en crear soluciones de datos que generen un impacto comercial medible.

Proyectos de Datos y Python

Análisis de Ineficacia en Call Center y Pruebas de Hipótesis

Stack Tecnológico: [Python] [Pandas] [SciPy/Estadística] [Tableau]
El Desafío: Una empresa de telecomunicaciones necesitaba identificar a los operadores de call center con bajo rendimiento y comprender la causa raíz de la disminución en la calidad del servicio tras un pico masivo en el volumen de llamadas.
La Solución y el Impacto: Diseñé una métrica de ineficacia personalizada analizando más de 50,000 registros y realicé pruebas de hipótesis estadísticas (Pruebas T). Demostré que los agentes etiquetados con "bajo rendimiento" manejaban el mismo volumen de llamadas que los mejores, lo que resultó en una recomendación estratégica de re-capacitación en lugar de despidos, ahorrando a la empresa significativos costos de rotación de personal.
Enlaces: Repositorio en GitHub | Dashboard en Tableau

Pipeline de Detección de Anomalías para la Misión PLATO de la ESA

Stack Tecnológico: [Python] [Data Pipelines] [Procesamiento de Señales] [Modelado Estadístico]
El Desafío: La futura misión espacial PLATO de la Agencia Espacial Europea requería un método automatizado y de alta precisión para filtrar anomalías y "falsos positivos" de enormes flujos de datos de sensores fotométricos.
La Solución y el Impacto: Diseñé y validé un algoritmo robusto de detección de anomalías mediante procesamiento de señales. Esta metodología fue adoptada oficialmente en el pipeline de datos de la misión para garantizar la integridad de los datos posteriores y resultó en una publicación científica como primer autor.
Enlaces: Repositorio en GitHub | Publicación Oficial

Dashboard Interactivo de Análisis de Ventas de Automóviles (EDA)

Stack Tecnológico: [Python] [Pandas] [Streamlit] [Visualización de Datos] [Render]
El Desafío: Los stakeholders no técnicos necesitaban una forma intuitiva de explorar y extraer información valiosa de negocio a partir de un conjunto de datos complejo sobre ventas de automóviles en Estados Unidos.
La Solución y el Impacto: Desarrollé y desplegué una aplicación web interactiva que permite a los usuarios realizar un Análisis Exploratorio de Datos (EDA) en tiempo real, mediante el uso de filtros dinámicos y visualizaciones automatizadas.
Enlaces: Repositorio en GitHub | Aplicación Web

Paquete Open-Source de Detección Planetaria (platos_closet)

Stack Tecnológico: [Python] [PyPI Packaging] [NumPy] [Sphinx/ReadTheDocs]
El Desafío: La comunidad astronómica necesitaba una herramienta ligera y de fácil acceso para evaluar matemáticamente si ciertos cuerpos planetarios cumplían con los umbrales de detección de señal a ruido de la misión PLATO.
La Solución y el Impacto: Desarrollé, empaqueté y publiqué un paquete de Python de código abierto en PyPI (completamente documentado). La herramienta permite a los investigadores evaluar de forma limpia la detectabilidad de los planetas a través de un simple pip install.
Enlaces: Repositorio en GitHub | PyPI | Documentación Oficial

Investigación Académica

Instrumentación Espacial y Exoplanetas

Antes de hacer la transición al análisis de datos corporativo, mi carrera académica estuvo dedicada a la misión espacial PLATO de la Agencia Espacial Europea y al estudio de planetas extrasolares. Mi investigación se centró en el desarrollo de pipelines de datos para modelar discos protoplanetarios, caracterizar planetas de Período Ultra Corto (USP) y construir algoritmos de corrección para filtrar falsos positivos de curvas de luz fotométricas complejas.
