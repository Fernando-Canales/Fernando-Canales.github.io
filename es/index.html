Investigación

Fotometría de doble apertura de PLATO
La fotometría de apertura se utiliza ampliamente en telescopios espaciales como una forma de extraer curvas de luz de tránsito. Para la misión espacial PLATO, muchos objetivos tendrán sus curvas de luz extraídas a bordo. Sin embargo, usar una sola apertura no es suficiente para ver si la curva de luz producida es realmente planetaria. Por lo tanto, usar una segunda apertura podría ser útil para distinguir la naturaleza de las curvas de luz para una parte considerable de las estrellas objetivo de PLATO. Mi trabajo durante el doctorado se dedicó a probar esta técnica. Desarrollé un simulador que crea curvas de luz realistas de PLATO con tránsitos planetarios y binarias eclipsantes contaminantes. Luego apliqué varios algoritmos de aprendizaje automático para clasificar las curvas de luz producidas. Los resultados fueron muy prometedores, mostrando que esta técnica funciona.
