About Me

I am a Data Analyst and Python Developer with a PhD in Astrophysics. I specialize in extracting actionable business insights from massive, messy datasets using Python, advanced statistical testing, and modern data visualization tools.

During my recently completed PhD between the Paris Observatory and the Max Planck Institute, I developed mission-critical anomaly detection algorithms for the European Space Agency's PLATO space mission. This rigorous academic environment ingrained in me a deep ability to solve highly ambiguous problems, engineer robust data pipelines, and write production-ready code.

I am now bringing this analytical firepower to the corporate tech sector. Whether it is building interactive Tableau dashboards, developing open-source Python packages, or using A/B testing to optimize business operations, my focus is on building data solutions that drive measurable impact.

Data & Python Projects

Anomaly Detection Pipeline for ESA's PLATO Mission

Tech Stack: [Python] [Data Pipelines] [Signal Processing] [Statistical Modeling]
The Challenge: The European Space Agency's upcoming PLATO mission required a highly accurate, automated method to filter false-positive anomalies out of massive streams of photometric sensor data.
The Solution & Impact: Engineered and validated a robust anomaly detection algorithm. This methodology was officially adopted into the mission's data pipeline to ensure downstream data integrity and resulted in a first-author publication.
Links: GitHub Repo | Publication

Interactive US Car Sales EDA Dashboard

Tech Stack: [Python] [Pandas] [Streamlit] [Data Visualization] [Render]
The Challenge: Non-technical stakeholders needed a user-friendly way to explore and extract business insights from a complex dataset of US automobile sales.
The Solution & Impact: Developed and deployed an interactive web application that empowers users to perform real-time Exploratory Data Analysis (EDA) through dynamic filtering and automated data visualizations.
Links: GitHub Repo | Live App

Open-Source Planetary Detection Package (platos_closet)

Tech Stack: [Python] [PyPI Packaging] [NumPy] [Sphinx/ReadTheDocs]
The Challenge: Astronomers needed a lightweight, easily accessible tool to evaluate if specific planetary bodies meet the signal-to-noise detection thresholds of the PLATO space mission.
The Solution & Impact: Developed and published a fully documented, open-source Python package to PyPI. The tool allows researchers to seamlessly calculate transit depths and assess detectability via a simple pip install.
Links: GitHub Repo | PyPI | Docs

Call Center Inefficiency Analysis & Hypothesis Testing

Tech Stack: [Python] [Pandas] [SciPy/Statistics] [Tableau]
The Challenge: A telecom company needed to identify underperforming call center operators and understand the root cause of declining service quality amidst a massive spike in call volume.
The Solution & Impact: Engineered a custom inefficiency metric across 50,000+ records and conducted statistical hypothesis testing (T-tests). Proved that underperforming agents handled the same call volume as top performers, resulting in a data-driven recommendation to retrain rather than fire staff—saving significant HR turnover costs.
Links: GitHub Repo | Tableau Dashboard

Academic Research

Space Instrumentation & Exoplanets

Before transitioning to corporate data analytics, my academic career was dedicated to the European Space Agency's PLATO mission and the study of extrasolar planets. My research focused on developing data pipelines to model protoplanetary disks, characterize Ultra Short Period (USP) planets, and build correction algorithms to filter false positives out of complex photometric light curves.
